# 헬로tv모바일 APP

(Feat. 휴대폰으로 보는 TV)

  • 매월 최신 VOD 최대 30편 무료제공
  • 실시간 방송 무료제공(최대60개채널)
  • 4명까지 함께 이용가능
  • 원격녹화 기능(UHD RED상품가입)

# 헬로tv모바일 연결하기

  1. 1. 헬로tv모바일 APP 다운받기

    스토어에서 “헬로tv”를 검색하여 APP을 다운받으세요

    2. 휴대폰 기기 등록하기

    리모콘의 “메뉴” > TV화면의 “안내/설정” > “스마트폰 연결” > “헬로tv APP 등록하기” 선택

    3. 휴대폰-TV 연동하기
    • “헬로tv APP 등록하기” 화면에서 내 휴대폰 번호 입력 (최대 4명까지 가능)
    • 내 휴대폰으로 전송되어 오는 “인증번호” 확인
    • TV에 인증번호 입력하면 연동 끝!
  4. 4. 연동 확인하기

    APP 하단에 현재 시청 중인 프로그램이 뜬다면 연결 완료!
    (혹시 프로그램이 뜨지 않는다면, 앱을 종료하고 재실행해보세요)

자, 연결까지 모두 되셨나요?
이제 다양한 콘텐츠를 즐기시면 되는데요,
다음 “이용하기”편에서는 다양한 콘텐츠를 이용하는 방법과
RED 가입 고객을 위한 원격 녹화 방법을
더욱 쉽게 설명해드릴게요.
